# 贡献指南

感谢你对本项目的兴趣！我们欢迎任何形式的贡献，包括但不限于：

- 报告 Bug
- 提出新功能建议
- 改进文档
- 提交代码修复或优化

## 📌 贡献流程

1. **Fork 本仓库**  
   点击 GitHub 页面右上角的 "Fork" 按钮，创建你的副本。

2. **克隆到本地**
   ```bash
   git clone git@github.com:your-username/spark-movielens-analysis.git
   cd spark-movielens-analysis
   创建特性分支
```bash
git checkout -b feat/your-feature-name
```
进行修改并提交
```bash

git add .
git commit -m "feat: add real-time chart visualization"
```
推送并发起 Pull Request
```bash

git push origin feat/your-feature-name
```
然后在 GitHub 上创建 PR，选择 main 为目标分支。
🧪 代码规范
使用 Python 3.8+ 语法
遵循 PEP 8 风格（可使用 black 或 autopep8 格式化）
新增功能需包含必要注释
修改核心逻辑请附带测试说明（即使无自动化测试）
📝 文档更新
若修改了功能或配置，请同步更新 README.md。

❓ 问题反馈
请先在 Issues 中搜索是否已有类似问题。若无，请清晰描述：

环境信息（OS、Python 版本等）
复现步骤
错误日志（如有）
再次感谢你的贡献！🎉
